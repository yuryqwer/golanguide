# golanguide
Golang 面试 &amp; 后端通用面试指南

推荐在线阅读：[golanguide.cn](https://golanguide.cn)

## 本地运行
本项目基于 [Docusaurus](https://docusaurus.io/)，一个网站静态页面生成工具。如果你把代码拉到本地自己运行，可以按下列步骤操作。

### 安装 node
确保 node 版本在 20.0 及以上（可以通过`node -v`命令检查）。你也可以使用 [nvm](https://github.com/nvm-sh/nvm) 管理本地多个不同的 node 版本。

### 安装项目依赖
在项目目录下运行`npm install`或者`yarn`。

### 启动本地开发服务器
在项目目录下运行`npm run start`或者`yarn run start`，会在本地启动一个开发服务器并且自动打开浏览器，大部分对项目文件的修改都会直接反映在浏览器中，无需重启服务器。
