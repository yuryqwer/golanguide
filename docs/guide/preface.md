# 前言
## 起源
本项目的想法是在我找工作期间诞生的，受到 [JavaGuide](https://javaguide.cn/) 这个项目的启发，我希望也能有一个类似的围绕 Golang 的网站，包含 Golang 以及后端所需的其他知识，求职者只需看这一个网站就够了。

在 Github 上有一个相似的[项目](https://github.com/mao888/golang-guide)，但并没有对应的网站，在国内访问、学习都相对不太方便。另外，哪怕无人问津，本项目至少也能作为自己个人整理的资料库，方便自己学习。

## 内容
在内容范围这一块，我初步拟定了以下几个方向：
- [ ] 软技能（怎么写简历、什么时候投简历、怎么谈薪资等）
- [ ] 技术（Go/缓存/数据库/消息队列/数据结构与算法/网络/操作系统/系统设计/分布式/高并发/高可用）
- [ ] 项目（项目实战，适合简历无项目可写，或者也可以根据岗位要求选择有针对性的项目写在简历上）
- [ ] 面经（具体某个岗位的面经，这个可能需要广大网友帮忙编写）
